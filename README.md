## 📖 Overview

**Projectile Simulator** is an interactive, real-time 3D ballistic simulation engine. Built to visualize complex kinematic equations, it offers a hands-on laboratory for understanding physics concepts through code.

Users can manipulate environmental variables such as **gravity**, **wind speed**, and **launch parameters** to observe how they affect a projectile's trajectory in a highly realistic 3D environment generated with **React Three Fiber**.

## ✨ Key Features

*   **🧪 Advanced Physics Engine**: Custom RK4 (Runge-Kutta 4th Order) integrator for laboratory-grade precision.
*   **🌪️ Environmental Controls**: Experiment with `Air Resistance`, `Wind Velocity`, and even `Alien Gravity` (Moon, Mars, etc.).
*   **📊 Real-time Analytics**: Live charting of kinematic data (Height vs Time, Velocity vs Time).
*   **🎥 Dynamic Camera Systems**:
    *   **Gunner View**: Aim down sights.
    *   **Bullet Cam**: Follow the projectile mid-flight.
    *   **Spectator Orbit**: Free 360° inspection.
*   **🎨 Glassmorphism UI**: A modern, clean interface built for usability and aesthetics.

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18+)
*   npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/hemasaivattikuti25/Projectile-simulator.git

# 2. Enter the directory
cd Projectile-simulator

# 3. Install dependencies
npm install

# 4. Start the simulation
npm run dev
```

| Key / Action | Function |
